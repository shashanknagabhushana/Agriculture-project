🌾 India's Agricultural Crop Production Analysis (1997–2021)
📖 Overview

India's Agricultural Crop Production Analysis (1997–2021) is an interactive data visualization project developed using Tableau and presented through a responsive web interface. The project analyzes agricultural production trends across different states, crops, seasons, and years to provide meaningful insights into India's agricultural performance.

The dashboard enables users to explore agricultural data interactively and understand production patterns through a collection of visualizations and data stories.

🎯 Objectives
Analyze agricultural crop production in India from 1997 to 2021.
Compare production across different states and crops.
Study agricultural yield trends over time.
Examine cultivated area and seasonal production patterns.
Present agricultural insights using interactive dashboards and storytelling.
📊 Key Performance Indicators (KPIs)

The dashboard includes two KPI cards that provide a quick overview of agricultural performance:

Total Production
Total Yield

These KPIs summarize the overall agricultural output and productivity during the study period.

📈 Dashboard Visualizations

The interactive dashboard consists of nine visualizations designed to provide a comprehensive analysis of India's agricultural production.

• Total Production & Total Yield

KPI cards displaying the overall agricultural production and total yield.

• Trend of Agricultural Yield (1997–2021)

Line Chart

Illustrates yearly changes in agricultural yield to identify long-term trends and variations.

• State-wise Agricultural Production Distribution

India Map

Displays agricultural production across different Indian states using a geographical map.

• Crop-wise Total Production Analysis

Horizontal Bar Chart

Compares total production among various crops cultivated across India.

• State-wise Cultivated Area Comparison

Grouped Bar Chart

Shows the cultivated area across different states for comparative analysis.

• Leading States by Agricultural Production

Bar Chart

Ranks states according to their agricultural production levels.

• Year-wise Total Agricultural Production (1997–2021)

Bar Chart

Represents annual agricultural production over the study period.

• Production Distribution by Unit Type

Donut Chart

Shows the proportion of agricultural production based on different unit types.

• Season-wise Agricultural Production Share

Pie Chart

Illustrates the percentage contribution of different agricultural seasons to total production.
